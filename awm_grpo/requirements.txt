# ── Core ML Framework ──────────────────────────────────────────────────────

# Or use the CUDA-specific URL:
#   pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128

# ── Flash Attention (tricky to install) ────────────────────────────────────
# flash-attn requires: torch, CUDA toolkit (nvcc), and a C++ compiler.
#
# Option A (recommended): Install pre-built wheel
#   pip install flash-attn --no-build-isolation
#
# Option B: If Option A fails, build from source
#   pip install packaging ninja
#   pip install flash-attn --no-build-isolation --no-cache-dir
#
# Option C: Use the slime Docker image which has it pre-installed
#
# Common errors:
#   - "No CUDA runtime found" → install cuda-toolkit: conda install cuda-toolkit -c nvidia
#   - "nvcc not found" → export PATH=/usr/local/cuda/bin:$PATH
#   - Build takes 20-40 min → use pre-built wheel or Docker
